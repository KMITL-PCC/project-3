# 🏫 Rooms API (WIP)

> **Status:** 🚧 Under Development

`Base URL: /api/rooms`

| Method | Path | Description |
|--------|------|-------------|
| `GET` | `/rooms` | ดึงข้อมูลห้องทั้งหมด |
| `GET` | `/rooms/:id` | ดึงข้อมูลห้องตาม ID |
| `POST` | `/rooms` | เพิ่มห้องใหม่ |
| `PUT` | `/rooms/:id` | แก้ไขข้อมูลห้อง |
| `DELETE` | `/rooms/:id` | ลบห้อง |
